🎓 EduTrack – Student Management & Attendance Tracking System

EduTrack is a modern, full-featured Student Management and Attendance Tracking System designed for schools, colleges, and training institutes.
It simplifies student administration by providing an intuitive dashboard for teachers and a clean, minimal interface for students.

Built using Vue.js, Node.js, and MongoDB, EduTrack delivers speed, clarity, and reliability.

🚀 Key Features
🧑‍🏫 Teacher Dashboard

Manage Courses and Subjects

Add, edit, delete students

Mark and update attendance

View detailed attendance records

View student-wise attendance percentages

Search students within a selected course

Pop-up action menus for clean UI

Attendance statistics with Pie & Bar charts

🎓 Student Dashboard

View personal profile

Check daily attendance

View attendance percentage

Clean, distraction-free UI (only Logout button visible)

🛠️ Tech Stack
Frontend

Backend
